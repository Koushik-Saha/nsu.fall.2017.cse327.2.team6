The Software Design Specification Outline

1. Introduction


    1. Purpose

This document will define the design of the KORBANI ANIMAL PROVIDER. It contains specific information about the animals and their price, age, weight (approximate), little description. The interaction between the classes to meet the desired requirements are outlined in detailed figures at the end of the document.


    1. Scope


This Design Specification is to be used by Software Engineering and Software Quality Engineering as a definition of the design to be used to implement our project KORBANI ANIMAL PROVIDER.

The KORBANI ANIMAL PROVIDER system is composed of two components one is user and another is admin. Admin can upload every animal picture with details specification and user can sign up with name, username, email address, phone number and the location. User get own profile where he/she can update delete and modify his/her profile. User can chose any kinds of animal with any range there is a range of money where user define what range he/she buy an animal. One animal have multiple buyer those case there has an auction who win this auction he/she can buy this animal.


    1. Definitions, acronyms, and abbreviations



Term

Definition

API

Application Program Interface. An API is a method by which an application program (a complete program that performs a specific function directly for the user) can access the computer's operating system.

GUI

Graphical User Interface. The GUI provides a graphical interface for users to interact with the system.

Link

A link is a means of connection between screens. Note: This is not short for “Hyperlink.”

PDA

Personal Digital Assistant. A PDA is a small, portable, computer-like device that can run software applications.

SDK

Software Development Kit. A set of programs that allows software developers to create products to run on a particular platform or to work with an API.

Synchronization

Synchronization is the act of connecting the Personal Digital Assistant to a computer, using a special device called a docking station, whereby the information in both the PDA and the computer become identical.
